Question-1: What are the different ways to select an element in the DOM?
There are several ways to select an element. Such as -getElementById,getElementByClass,getElementByTag,querySelector etc.
Question-2: What is the difference between innerHTML, innerText, and textContent?
By using 'innerHTML' you can trigger entire html content
By using 'innerText' you can trigger a specific text inside an element and set or change it.
'textContent'represents the text content of an element, including child elements without any HTML tags.
Question-3: What is event delegation in the DOM?
By using it you can add event listener directly to parent element instead of using event listener for multiple times for multiple elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event starts from the root and keep traveling through all the child elements and then goes back to the root again.
Question-5: How do you create, add, and remove elements using JavaScript?
Creating an element: document.createElement('')
Adding an element: appendChild()
removing an element: remove()
